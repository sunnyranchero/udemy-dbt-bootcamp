# dbt Capstone Project: AirStats
Congratulations on making it this far in the course! It's time to put your dbt skills to the test with a hands-on capstone project.

# What is it?
The AirStats Capstone Project is a complete, end-to-end dbt assignment where you'll build a data pipeline analyzing global airport data on Snowflake. You'll work with real-world aviation data from [OurAirports.com](https://ourairports.com/data/) — covering ~72,000 airports, ~44,000 runways, and user-submitted airport comments.


# What you'll practice
- Defining **sources** and **staging models** (bronze layer)
- Building **silver layer** tables with data cleaning and transformations
- Creating an **incremental model**
- Implementing **snapshots** (SCD Type 2 with check strategy)
- Writing **tests** (generic, singular, and dbt-expectations)
- Adding **documentation** with `doc()` blocks and an overview page

# Important info for students who started the course earlier
